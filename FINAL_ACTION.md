## 🆘 If Something Goes Wrong

1. **Still showing error after rebuild?**
   - Clear browser cache (Ctrl+Shift+Delete)
   - Try in incognito/private mode
   - Wait 1 more minute (DNS propagation)

2. **Build failed?**
   - Check the build logs in Render dashboard
   - Common fix: Click "Manual Deploy" again

3. **Can't login to Render?**
   - Use the email you used to sign up
   - Render Dashboard: https://dashboard.render.com

4. **API not connecting?**
   - Open browser console (F12)
   - Check for CORS errors
   - Backend status: https://task-manager-api-25g1.onrender.com/tasks
   - Should show: `[]` (empty list)
