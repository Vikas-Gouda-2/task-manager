## 💾 DATABASE INFORMATION

### Current: SQLite
- File-based database
- Perfect for development
- Data stored in: `backend/tasks.db`
- All tables auto-created

### Schema:

**user_profile** - Gamification data
- id, xp, level, streak_days

**tasks** - Task management
- id, title, category, priority, due_date, completed, xp_reward, notes, tags

**subtasks** - Task breakdown
- id, task_id (FK), title, completed

### Production Ready: PostgreSQL
- Can upgrade anytime
- One environment variable change
- Data persists permanently
